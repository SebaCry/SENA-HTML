GLOSARIO DE TERMINOS

- HTML:
Es una etiqueta que define el fin y el comienzo del documento HTML ademas de su estructura Mas informacion
- H1:
Es una etiqueta que define los elementos del encabezado ,h1 es el mas importante y mas grande y h6 es el mas pequeño pero sirven para titular los parrafos Mas informacion
- A:
Es una etiqueta que enlanza a otras paginas, ubicaciones o archivos dentro del mismo archivo Mas informacion
- P:
Define un párrafo de texto, útil para organizar contenido en bloques. Mas informacion
- DIV:
Contenedor genérico para agrupar otros elementos, usado para diseño y estilos Mas informacion
- UL:
Este crea la lista, donde define que los elementos serán en viñetas Mas informacion
- LI:
Se crean los elementos de la lista dependiendo a los atributos de ella Mas informacion
- FORM:
Crea un formulario para recoger datos del usuario, como texto o correos. Mas informacion
- SCRIPT:
Inserta o enlaza código JavaScript para agregar interactividad a la página. Mas informacion
- LINK:
Enlaza recursos externos, como iconos u hojas de estilo. Mas informacion
